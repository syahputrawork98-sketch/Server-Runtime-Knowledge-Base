# SR-02: V8 Bindings - JavaScript to C++ Bridge

Membahas bagaimana V8 diintegrasikan ke dalam Node.js dan bagaimana objek JavaScript dipetakan ke fungsi C++.

## 🎯 Misi Pembelajaran
- Memahami konsep V8 Isolates, Contexts, dan Handles.
- Membedah mekanisme penulisan "Native Abstractions" sebelum N-API.
- Memahami delegasi tugas dari JS ke mesin internal C++.
