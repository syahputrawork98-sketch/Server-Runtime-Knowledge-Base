# Referensi Utama (Buku Besar) - Server Runtime

Dokumen berikut dianggap sebagai **buku besar** atau **referensi induk** untuk Server Runtime (fokus utama: Node.js, Libuv, V8).

## Core Runtime & Engine

- **Node.js Official Documentation**: Sumber utama untuk API dan perilaku runtime.
- **Libuv Documentation**: Referensi untuk abstraksi I/O asinkron dan event loop.
- **V8 Embedder's Guide**: Dokumentasi internal V8 untuk integrasi C++/JS.
- **v8.dev**: Blog dan dokumentasi arsitektur mesin V8.

- **RFC 6455 (The WebSocket Protocol)**.
- **ECMAScript Specification (ECMA-262)**.
- **Node-API (formerly N-API) Documentation**.

## Sistem Operasi & I/O

- **The Linux Programming Interface** (Michael Kerrisk).

## Dokumentasi Tambahan
